MODELANDO CON TECNOLOGIA 3D SOLO CON CSS

CUBO

Este cubo 3D está construido completamente con CSS utilizando transform-style: preserve-3d y animaciones para lograr un efecto de rotación continua. Cada una de sus seis caras tiene un color distinto y está posicionada correctamente en el espacio tridimensional con rotateX(), rotateY() y translateZ(). Además, la perspectiva se maneja mediante un contenedor .perspective para que la profundidad sea más realista.

El cubo puede ser utilizado como ejemplo de modelado 3D en CSS sin necesidad de JavaScript o librerías externas. La animación lo hace girar de manera fluida, permitiendo visualizar todas sus caras en movimiento.
